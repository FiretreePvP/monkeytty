# 🦀 Rust Typing Test for the terminal (monkeytty)

It's my first Rust project.
It is a simple typing test implemented in Rust. The goal is to measure typing speed and accuracy while rewriting a given piece of text.

## 🚀 Goal
In the long term, this project aims to become a small application to casually help users improve their typing skills. The main focus is on:

- Speed  
- Accuracy  
- Ease of use  

## 📌 Status
🚧 Work in progress

- A very basic I/O implementation with one word at a time.
- The real features are currently being implemented.

## ✨ Planned Features
- Infinite text
- Measurement of typing speed
- Detection and counting of typing errors
- Accuracy evaluation

## 🛠️ Technologies
- Rust
- Terminal-based user interface (Ratatui planned)

## 📦 Installation
- [Install Rust](https://rust-lang.org/tools/install/)

```bash
git clone https://github.com/FiretreePvP/monkeytty.git
cd monkeytty
cargo run
```
